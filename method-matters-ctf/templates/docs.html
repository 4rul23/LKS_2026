Transaction Export API

Programmatic access to transaction data export functionality.

/api/v1/transactions/export

Description

Export transaction records for reporting and reconciliation purposes. Access is restricted based on authentication level and request method.

Authentication

This endpoint requires proper authentication. Include your API key in the request header:

Authorization: Bearer <your_api_key>

Response Codes

200 Success - Data exported
401 Unauthorized - Invalid or missing credentials
403 Forbidden - Access denied
405 Method Not Allowed

Example Request

# Export transactions

curl -H "Authorization: Bearer $API_KEY" \

http://api.nexuspay.io/v1/transactions/export
