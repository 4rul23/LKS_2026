Transaction Export API

Modul untuk ekspor data transaksi history.

REST
/api/v1/transactions/export

Deskripsi

Endpoint ini digunakan untuk mengunduh laporan transaksi harian dalam format JSON. Akses dibatasi sesuai level otorisasi pengguna.

Security Note (UU PDP Compliance): Permintaan GET standar telah dinonaktifkan. Client diharapkan menggunakan metode HTTP yang sesuai dengan standar internal kami.

Autentikasi

Endpoint ini membutuhkan API Key yang valid pada header:

Authorization: Bearer <your_api_key>

Response Codes

200 OK - Data berhasil diekspor
401 Unauthorized - Token tidak valid
403 Forbidden - Akses metode GET ditolak
405 Method Not Allowed

Contoh Request (cURL)

# Export transactions data

curl -X [METHOD] \

-H "Authorization: Bearer $API_KEY" \

http://api.nusanta.ra/v1/transactions/export
